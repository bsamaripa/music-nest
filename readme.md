Music Nest
=========

Analyzes a given users input for interactions between other users and returns potential leaders within that social group.

Team
--------
* Byron Samaripa
* Landon Bedell

Requirements
------
Nodejs
mongodb

Installation
--------
```sh
git clone git@github.com:bsamaripa/music-nest.git
npm install
bower install
node server.js
```
